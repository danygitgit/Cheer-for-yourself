Cheer-for-yourself
===

&emsp;本文档库始建于 `2018-12-13`，目的是做学习笔记。  

&emsp;梁sir创建了一个 “ Cheer-for-yourself ” 活动，旨在互相监督学习，促进成长。很荣幸可以加入到这个活动中来。

&emsp;作为一个前端搬砖工，希望以此为平台，记录自己的成长历程，向大佬学习。

&emsp;路漫漫其修远兮！与出君共勉，为自己干杯！
